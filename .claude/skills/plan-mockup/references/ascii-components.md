# ASCII 와이어프레임 컴포넌트 라이브러리

Plan mode에서 UI 레이아웃을 시각화하기 위한 표준 컴포넌트.
모든 컴포넌트는 **65자 폭** 기준 (Rule 11 준수).

---

## 기본 규칙

- 단선: `┌─┐└─┘│` (일반 영역)
- 겹선: `╔═╗╚═╝║` (강조/포커스 영역)
- 플레이스홀더: `[컴포넌트명]`, `░░░` (텍스트 자리)
- 상호작용: `(click)`, `(submit)`, `(hover)` 주석
- 데이터: `{변수명}` 동적 값 표시

## 상호작용 주석 규칙

와이어프레임 아래에 상호작용을 명시:

```
(click) [User ▼] → Profile Dropdown
(click) Analytics → analytics-detail screen
(submit) Form → POST /api/data
(hover) Card → Tooltip with details
```
