2-7 Single Draw vs Triple Draw — 교환 횟수만 다르다

Triple Draw (3회)

5장 → 1차 교환 → 베팅 → 2차 교환 → 베팅 → 3차 교환 → 쇼다운

기회 3번 — 여유롭게 개선
VS
Single Draw (1회)

5장 → 1회 교환 → 베팅 → 쇼다운
