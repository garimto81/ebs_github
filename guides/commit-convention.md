# 커밋 메시지 규칙

## 형식

```
<type>: <설명>
```

## Type

| type | 용도 |
|------|------|
| `docs` | 기획 문서 추가/수정 |
| `fix` | 오류/오타 수정 |
| `backlog` | 백로그 생성/수정 |
| `guides` | 가이드 문서 추가/수정 |
| `chore` | 설정, 도구, 기타 |

## 예시

```
docs: Omaha Hi-Lo 규칙 추가
docs: PoC 시나리오 v2.0 업데이트
fix: 베팅 시스템 Ante 설명 오타 수정
backlog: 방송 셋업 Epic 생성
backlog: 딜링 Epic Story 3건 추가
guides: 버저닝 규칙 문서화
chore: .gitignore 설정
```

## 규칙

- 한글 또는 영문 모두 가능
- 설명은 간결하게 — 무엇을 했는지 한 줄로
- 본문(body)은 선택 사항, 필요할 때만 추가
