[19]
STRIP STACK Button — STRIP STACK 전송. 전체 플레이어 스택 비교 스트립 그래픽 방송 전송
[20]
STRIP WIN Button — STRIP WIN 전송. 전체 플레이어 수익 비교 스트립 그래픽 방송 전송
[21]
TICKER Button — TICKER 그래픽 전송. 하단 흐르는 텍스트(ticker) 그래픽 방송 전송
[22]
Arrow Right Button — 다음 화면 또는 추가 컨트롤로 이동하는 네비게이션 버튼
