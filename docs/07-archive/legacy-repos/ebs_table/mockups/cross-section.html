EBS RFID Poker Table — Build Guide V2.2

EBS 피처 테이블 단면도
5-piece MDF 적층 구조

수직 단면 (테이블 중앙 기준)  ·  수직 축척 약 5:1 과장

18mm MDF (Base, Rail-bot)
12mm MDF (Racetrack, Rail-top)
9mm MDF (Cutout)
Open Cell Foam (12mm, rail exterior)
Closed Cell Foam (5mm, playing surface)
Vinyl / Speed Cloth (~1mm)
Antenna (14mm routed cavity in Base)

축척 주의: 수직 축척을 약 5:1로 과장하여 얇은 레이어(비닐, 속클로스)를 가시적으로 표현했습니다. 수평 방향은 약 1:4 축소. 실제 치수는 각 레이어 레이블의 mm 표기를 기준으로 하십시오. 단면 위치: 테이블 장축 중앙 수직 절단면 기준.
