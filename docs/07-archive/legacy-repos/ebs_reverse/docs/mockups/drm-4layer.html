4-Layer DRM System Architecture

Layer 1: Email/Password Login
Email/Password → LoginCommand → FluentValidation → AuthenticationService → RemoteLoginResponse
(Token, ExpiresIn)
Layer 2: Offline Session
OfflineSessionService → OfflineLoginEntry
(암호화 저장) →
OfflineLoginStatus (Success / Failure / Expired / Found / NotFound)
Layer 3: KEYLOK USB Dongle
DongleService → KeylokDongle
(47+ KEYLOK API) →
KLCHECK, READAUTH GETSN, GETEXPDATE LaunchAntiDebugger
Layer 4: Remote License Server
LicenseBackgroundService (5분 간격 HTTP) → DeviceData
(DongleType + SerialNumber + GlobalId) → RemoteLicenseResponse
(License + Signature)
