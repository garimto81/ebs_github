Dependency Tree Architecture

vpt_server.exe Application Layer (Entry Point)
mmr.dll GPU/Video Engine
SharpDX, MFORMATSLib, System.Drawing
PokerGFX.Common.dll Shared Platform
DI Container, Configuration, Cryptography
net_conn.dll Network Protocol Layer
hand_eval.dll Poker Hand Evaluation Engine
RFIDv2.dll RFID Hardware Interface
boarssl.dll BearSSL TLS Encryption
analytics.dll Telemetry & Monitoring
AWS S3 Upload, Metrics Collection
Architecture Layers:
• Application Layer: vpt_server.exe (메인 애플리케이션)
• Core Services: mmr, net_conn, hand_eval (핵심 기능)
• Platform Layer: PokerGFX.Common (공통 인프라)
• Hardware Interface: RFIDv2 + boarssl (하드웨어 통신)
• External Services: analytics (텔레메트리)
