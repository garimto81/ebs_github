4-Layer Protocol Stack

Layer 4
Application Layer
JSON (v2.0+) / CSV (v1.x Legacy)
게임 상태, 플레이어 액션, 그래픽 명령 전송
Layer 3
Encryption Layer
AES-256 (Rijndael, PBKDF1)
데이터 암호화 및 무결성 보장
Layer 2
Framing Layer
SOH (0x01) Delimiter
메시지 구분자, 프레임 경계 정의
Layer 1
Transport Layer
TCP (port 9001) + UDP (port 9000)
네트워크 전송, 신뢰성 보장 (TCP) / 실시간 전송 (UDP)
