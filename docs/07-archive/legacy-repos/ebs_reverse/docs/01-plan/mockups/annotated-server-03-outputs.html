Outputs 탭 기능 영역

1Video Size / Frame Rate — 출력 해상도 (1920x1080 등), 9:16 세로 모드 옵션, 프레임레이트 설정
2Live 출력 설정 — 실시간 Video Preview, Audio Preview, Output Device 선택
3Delay 출력 설정 — 지연 방송용 Video Preview, Audio Preview, Output Device 선택
4Key & Fill 체크박스 — Live/Delay 별도 Key & Fill 출력 모드 (외부 합성기 연동)
5Virtual Camera — NDI/가상 카메라 출력 활성화
6Security Delay — 보안 지연 시간(분) 설정 (홀카드 노출 방지)
7Dynamic Delay / Auto Stream — 동적 지연 조절 및 자동 스트리밍 시작 타이머
8Countdown 설정 — 카운트다운 표시, Lead-Out 비디오, 배경 이미지 설정
9Recording Mode — 녹화 모드 (Video with GFX, Video only 등) 선택
10Twitch Account 연동 — Twitch 계정 연결, ChatBot, Channel Title 설정
