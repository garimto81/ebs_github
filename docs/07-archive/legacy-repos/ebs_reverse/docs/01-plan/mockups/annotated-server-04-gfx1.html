GFX 1 탭 기능 영역

1Board Position / Player Layout — 보드 카드 위치(Left/Right/Centre), 플레이어 박스 레이아웃(Vert/Bot/Spill) 선택
2Reveal Players / Reveal Cards — 플레이어/카드 공개 시점 설정 (Action On, Immediate 등)
3Fold / Transition 설정 — 폴드 표시 방식, Transition In/Out 애니메이션 종류 및 시간
4Heads Up 설정 — 헤즈업 레이아웃 Left/Right, 카메라 위치, Custom Y pos
5스킨 정보 + Skin Editor — 현재 적용된 스킨 이름/용량 표시, Skin Editor 실행 버튼
6Media Folder — 미디어 리소스(스폰서 로고, 이미지) 폴더 열기
7스폰서 로고 3개 — Leaderboard/Board/Strip 위치별 스폰서 로고 이미지 등록
8Vanity 테이블 이름 — 화면에 표시할 테이블 이름 + Replace Vanity with Game Variant 옵션
9마진 설정 — X Margin, Top Margin, Bot Margin (%) 미세 조정
10추가 옵션 — Heads Up History, Indent/Bounce Action Player, Leaderboard/PIP/Stats after hand, Action Clock
