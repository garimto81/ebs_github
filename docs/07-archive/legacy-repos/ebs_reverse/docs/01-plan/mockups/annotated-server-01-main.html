Main Window 기능 영역

1GPU 프리뷰 영역 — 실시간 합성된 그래픽 오버레이를 미리 볼 수 있는 메인 캔버스
2CPU / GPU 상태 표시등 — 하드웨어 인코더 활성 상태를 색상으로 표시
3Secure Delay 체크박스 — 방송 보안 지연 활성화 (홀카드 노출 방지)
4Preview 체크박스 — GPU 프리뷰 렌더링 ON/OFF 토글
5Reset Hand 버튼 — 현재 핸드 상태 초기화 (설정/잠금 아이콘 포함)
6Register Deck 버튼 — RFID 카드 덱 등록 시작
7Launch Action Tracker 버튼 — 별도 Action Tracker 창 실행
8Studio 버튼 — 스튜디오 모드 전환
9Split Recording 버튼 — 녹화 파일 분할
10Tag Player 버튼 — 플레이어 태그 지정 (이름표 입력란 포함)
11경고/잠금 아이콘 — 시스템 경고 및 테이블 잠금 상태 표시
