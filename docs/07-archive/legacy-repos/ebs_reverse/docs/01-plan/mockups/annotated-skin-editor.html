Skin Editor 기능 영역

1Name / Details 입력 — 스킨 이름과 설명 텍스트 입력
24K + Transparency 옵션 — Designed for 4K (3840x2160) 체크, Chroma Key 시 Partial Transparency 제거
3Adjustments — Adjust Size 슬라이더 및 색상 조절 안내 (Element 선택 후 Adjust Colours 클릭)
4Elements — Board, Blinds, Outs, Strip, Hand History, Action Clock, Leaderboard, Split Screen Divider, Ticker, Field 9개 요소 편집 버튼
5Cards — 카드 이미지 프리뷰 (6벌), Add/Replace/Delete 버튼, Import Card Back
6Flags — Country flag 모드, Edit Flags 버튼, Hide flag after 시간 설정
7Text — Font 1/2 선택, Text All Caps 체크, Text Reveal Speed 슬라이더, Language 설정
8Player — Variant (HOLDEM 2 Cards 등) 선택, Player Set (2 Card Games 등), Override Card Set, Edit/New/Delete, Crop photo to circle
9하단 버튼바 — IMPORT, EXPORT, SKIN DOWNLOAD CENTRE, RESET TO DEFAULT, DISCARD, USE
