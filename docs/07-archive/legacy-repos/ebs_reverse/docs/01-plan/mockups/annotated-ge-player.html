Graphic Editor (Player) 기능 영역

1Player Set 드롭다운 — 편집할 플레이어 세트 선택 (2 Card Games 등)
2캔버스 크기 — Player 요소의 캔버스 크기 표시 (465 X 120)
3Import Image / AT Mode — 배경 이미지 가져오기 및 AT Mode with photo
4Element 드롭다운 — 편집할 요소 선택 (Card 1, Name, Action, Stack, Position 등)
5Transform 패널 — Left, Top, Width, Height, Z-order, Anchor 위치 및 크기 설정
6Animation 패널 — AnimIn/AnimOut, Transition In/Out 설정
7Text 패널 — Font, Alignment, Drop Shadow, Colour/Hilite Colour, Rounded Corners, Margins
8Background Image + Adjust Colours — 요소 배경 이미지 및 색상 조정, OK/Cancel 버튼
9WYSIWYG 프리뷰 — 플레이어 박스 실시간 미리보기 (사진, 카드, NAME, ACTION, STACK, POS, 국기, Equity)
