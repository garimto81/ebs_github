System 탭 기능 영역

1Table 설정 — 테이블 Name과 Password 입력 (각각 Update 버튼)
2Reset / Calibrate — RFID 리더 초기화 및 캘리브레이션
3License 정보 — Serial 번호, 라이선스 상태 표시
4Check for Updates — 업데이트 확인, Updates & support / PRO license Evaluation mode 버튼
5MultiGFX / Sync 옵션 — MultiGFX, Sync Stream, Sync Skin, No Cards, Disable GPU Encode, Ignore Name Tags
6RFID/AT 관련 옵션 — UPCARD antennas, Disable muck antenna, Community Card antenna, Auto Start, Allow AT access, AT Predictive Bet, AT Kiosk
7시스템 진단 + 하드웨어 정보 — Open Table Diagnostics 버튼, CPU/GPU/OS/Encoder 정보 표시
8View System Log — 시스템 로그 뷰어 실행
9Secure Delay Folder / Export Folder — 보안 지연 녹화 및 내보내기 폴더 경로 설정
10Stream Deck 설정 — Elgato Stream Deck 연동 활성화
