Graphic Editor (Board) 기능 영역

1캔버스 크기 — Board 요소의 캔버스 크기 표시 (296 X 197)
2Import Image / AT Mode — 배경 이미지 가져오기 및 Action Tracker 모드 (Flop Game) 선택
3Element 드롭다운 — 편집할 요소 선택 (Card 1, Card 2, ... POT, Table Name 등)
4Transform 패널 — Left, Top, Width, Height, Z-order, Anchor (Right/Top 등) 위치 및 크기 설정
5Animation 패널 — AnimIn/AnimOut 활성화 (X 체크), Transition In/Out 종류 (Default 등)
6Text 패널 — Text Visible 체크, Font 선택, Alignment, Drop Shadow, Colour/Hilite Colour, Rounded Corners, Margins
7Background Image — 요소 배경 이미지 추가/제거 (Click to add)
8Adjust Colours — 선택된 요소의 색상 미세 조정 대화상자 열기
9WYSIWYG 프리뷰 — Board 카드(A,2,3,4,5), POT, 블라인드, 테이블 이름을 실시간 미리보기
