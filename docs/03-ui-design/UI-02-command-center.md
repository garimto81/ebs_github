## 개요

Command Center(CC)는 운영자가 포커 핸드를 실시간 진행하는 Flutter 앱이다. 테이블당 1개 인스턴스로 동작하며, 운영자 주의력의 85%가 집중되는 핵심 화면이다.

> 참조: BS-05-00-overview, BS-05-02-action-buttons, BS-00 §1

---

## CC 레이아웃 3영역

모든 화면에서 상단 바는 고정이다. 중앙 영역만 화면별로 변경된다.

```
+-------------------------------------------------+
| [Status Bar] Game | Hand # | RFID | BO    48px |
+-------------------------------------------------+
|                                                 |
|              [Content Area]                     |
|             (화면별 변경)                        |
|                                                 |
+-------------------------------------------------+
| [Action Panel] 8 Buttons + UNDO            80px |
+-------------------------------------------------+
```

## 상단 바 (공통)

```
+-------------------------------------------------+
| ● BO  ● RFID | HOLDEM | Hand #42 | 100/200  [⚙]|
+-------------------------------------------------+
```

| 항목 | 위치 | 바인딩 |
|------|------|--------|
| BO 연결 | 좌측 | ● 녹색 / ○ 빨간색 |
| RFID 상태 | 좌측 | ● 녹색 / ○ 회색 / ⚠ 빨간색 |
| Game Type | 중앙좌 | `game_type` enum |
| Hand # | 중앙 | `hand_number` 자동 증가 |
| Blinds | 중앙우 | `SB/BB` |
| [Settings ⚙] | 우측 | Admin 전용 |

## 화면 1: 메인 화면 (테이블 + 액션)

CC의 기본 화면. 핸드 진행의 모든 것이 여기에 표시된다.

```
+-------------------------------------------------+
| ● BO  ● RFID | HOLDEM | Hand #42 | 100/200  [⚙]|
+-------------------------------------------------+
|          [S6]    [S7]    [S8]                   |
|       [S5]                  [S9]                |
|                                                 |
|       [S4]   [Board Area]   [S0]                |
|                [POT: 2,400]                     |
|       [S3]                  [S1]                |
|          [S2]    [D]                            |
+-------------------------------------------------+
| NEW  DEAL FOLD CHECK BET CALL RAISE ALL-IN     |
| [N]  [D]  [F]  [C]   [B] [C]  [R]   [A]  UNDO |
+-------------------------------------------------+
```

### 좌석 위젯 (SeatWidget) 상세

각 좌석 [S0]~[S9]은 다음을 표시한다:

```
+---------------------------+
| John Doe            [D]  |
| Stack: 125,000           |
| [Ah] [Kd]   CHECK       |
| ████████░░ 82%           |
+---------------------------+
```

| 요소 | 바인딩 | 비고 |
|------|--------|------|
| 이름 | `seat.player.name` | 글자 수 초과 시 말줄임 |
| 포지션 뱃지 | D / SB / BB / STR | 해당 포지션에만 표시 |
| Stack | `seat.player.stack` | Roboto Mono, 콤마 포맷 |
| 홀카드 2장 | `seat.cards[0,1]` | RFID 감지 또는 수동 |
| 액션 배지 | `seat.last_action` | CHECK/FOLD/BET/CALL/RAISE/ALL-IN |
| Equity 바 | `seat.equity` | 0~100% 프로그레스 바 |

### action_on 표시

현재 액션 대상 좌석은 **glow 효과**(pulse 애니메이션)로 강조된다. 나머지 좌석은 일반 상태.

### Board Area

```
+-----------------------------------+
| [Flop1] [Flop2] [Flop3] [T] [R]  |
+-----------------------------------+
```

- 카드 슬롯 5개 (비어있으면 회색 placeholder)
- Flop: 3장 동시 공개
- Turn: 4번째 슬롯
- River: 5번째 슬롯

## 화면 2: 좌석 상세 패널

좌석을 탭하면 슬라이드업 패널이 표시된다.

```
+-------------------------------------------------+
|                  [메인 화면]                     |
|                 (반투명 배경)                    |
+-------------------------------------------------+
| +---------------------------------------------+ |
| | Seat 3 — Tom Lee                            | |
| |---------------------------------------------| |
| | Stack: 45,200      | Hands: 28              | |
| | VPIP: 32%          | PFR: 18%               | |
| | AGR: 2.1           | WTSD: 28%              | |
| |---------------------------------------------| |
| | Cards: [7h] [7d]   | Position: BB           | |
| |---------------------------------------------| |
| | [Adjust Stack] [Move Seat] [Remove] [Close] | |
| +---------------------------------------------+ |
+-------------------------------------------------+
```

### 구성 요소

| 요소 | 바인딩 | 동작 |
|------|--------|------|
| Player 이름 | `player.name` | 읽기 전용 |
| Stack | `player.stack` | [Adjust Stack]으로 수정 |
| 통계 4종 | `player.stats.*` | VPIP/PFR/AGR/WTSD |
| Cards | `seat.cards` | 현재 핸드 카드 |
| Position | `seat.position` | D/SB/BB/STR/UTG... |
| [Adjust Stack] | Button | 칩 수동 조정 다이얼로그 |
| [Move Seat] | Button | 좌석 이동 (드래그) |
| [Remove] | Button (danger) | 좌석에서 제거 |

## 화면 3: 카드 입력 (수동 / Mock)

RFID 미사용 또는 수동 폴백 시 카드를 직접 선택하는 화면.

```
+-------------------------------------------------+
| Card Input — Seat 3 (Hole Card 1/2)             |
+-------------------------------------------------+
|  ♠  | A  K  Q  J  T  9  8  7  6  5  4  3  2   |
|  ♥  | A  K  Q  J  T  9  8  7  6  5  4  3  2   |
|  ♦  | A  K  Q  J  T  9  8  7  6  5  4  3  2   |
|  ♣  | A  K  Q  J  T  9  8  7  6  5  4  3  2   |
+-------------------------------------------------+
| Selected: [Ah]  Remaining: 1                    |
| [Cancel]                          [Confirm]     |
+-------------------------------------------------+
```

### 구성 요소

| 요소 | 타입 | 동작 |
|------|------|------|
| 4x13 그리드 | CardGrid | 수트(행) x 랭크(열) |
| 사용된 카드 | 비활성 (회색) | 이미 딜된 카드 |
| Selected | CardImage | 선택된 카드 미리보기 |
| Remaining | Counter | 남은 선택 수 |
| [Confirm] | Button | `CardDetected` 이벤트 합성 |

### 터치 최적화

- 각 카드 셀 최소 48x48px 터치 타겟
- 수트 행 높이 56px
- 선택 시 즉각 시각 피드백 (배경 하이라이트)

## 화면 4: 금액 입력 (BET / RAISE)

BET 또는 RAISE 시 슬라이드업되는 금액 입력 패널.

```
+-------------------------------------------------+
|                  [메인 화면]                     |
+-------------------------------------------------+
| +---------------------------------------------+ |
| | BET Amount              Min: 200  Max: All  | |
| |---------------------------------------------| |
| | [MIN] [1/2 POT] [  POT  ] [ALL-IN]         | |
| |---------------------------------------------| |
| | [  |============●=====|  ]  Slider          | |
| |---------------------------------------------| |
| |      +------------------+                   | |
| |      |          2,400   |  Amount Field     | |
| |      +------------------+                   | |
| | [ 1 ][ 2 ][ 3 ]                            | |
| | [ 4 ][ 5 ][ 6 ]        [  BET  ]           | |
| | [ 7 ][ 8 ][ 9 ]        [Cancel ]           | |
| | [ C ][ 0 ][ <- ]                           | |
| +---------------------------------------------+ |
+-------------------------------------------------+
```

### 구성 요소

| 요소 | 타입 | 바인딩 |
|------|------|--------|
| Quick Preset | Button x4 | MIN / 1/2 POT / POT / ALL-IN |
| Slider | BetSlider | min ~ max 범위 |
| Amount Field | TextField (mono) | 직접 입력 가능 |
| Numpad | NumberPad 3x4 | 숫자 입력, C(초기화), <-(삭제) |
| [BET] / [RAISE] | Button (primary) | 금액 확정, 이벤트 전송 |
| [Cancel] | Button | 입력 취소, 메인으로 복귀 |

### 금액 검증

| 입력값 | 반응 |
|--------|------|
| < min_bet | 경고 + 재입력 |
| > max_bet | 경고 + 재입력 |
| > player.stack | 자동 ALL-IN 전환 |
| == 0 | 경고 ("0 베팅 불가") |

## 화면 5: Undo 히스토리

최근 이벤트 로그와 UNDO 기능.

```
+-------------------------------------------------+
| Undo History — Hand #42                         |
+-------------------------------------------------+
| #  | Time   | Event              | Player      |
|----|--------|--------------------|-------------|
| 5  | 14:32  | RAISE 1,200        | Seat 5      |
| 4  | 14:31  | CALL 400           | Seat 3      |
| 3  | 14:31  | BET 400            | Seat 0      |
| 2  | 14:30  | CHECK              | Seat 9      |
| 1  | 14:30  | DEAL               | System      |
+-------------------------------------------------+
| [UNDO Last] (Ctrl+Z)         [Close]           |
+-------------------------------------------------+
```

### 구성 요소

| 요소 | 바인딩 | 비고 |
|------|--------|------|
| Event Log | `hand.events[]` | 최신 이벤트 상단 |
| [UNDO Last] | Button | 최대 5단계 되돌리기 |
| 단축키 | Ctrl+Z | UNDO Last와 동일 |

## 화면 6: Hand History

과거 핸드 목록 조회.

```
+-------------------------------------------------+
| Hand History — Table 1                          |
+-------------------------------------------------+
| Hand | Winner      | Pot     | Board            |
|------|-------------|---------|------------------|
| #42  | Seat 0      |  2,400  | Ah Kd 7c 3s 9h  |
| #41  | Seat 5      |  1,800  | Fold pre-flop    |
| #40  | Seat 3,7    |  5,200  | Td 8d 2c Jd Qs  |
| #39  | Seat 1      |  3,000  | 9s 9h 4c Kc 2d  |
+-------------------------------------------------+
| [< Prev]  Page 1 of 5  [Next >]     [Close]    |
+-------------------------------------------------+
```

### 구성 요소

| 요소 | 바인딩 | 비고 |
|------|--------|------|
| Hand 목록 | `GET /tables/{id}/hands` | 페이지네이션 |
| Hand 번호 | `hand.number` | 클릭 시 상세 보기 |
| Winner | `hand.winners[]` | 복수 승자(Chop) 가능 |
| Pot | `hand.total_pot` | 콤마 포맷 |
| Board | `hand.board_cards` | 카드 이미지 또는 텍스트 |

## 화면 전환 플로우

```
  메인(1)
  |  |  \----→ 카드 입력(3)
  |  |  \----→ 금액 입력(4)
  |  |
  |  +--→ 좌석 상세(2) ──→ Adjust Stack
  |
  +----→ Undo 히스토리(5)
  +----→ Hand History(6)
  +----→ 통계 패널(7)
  +----→ Settings(8)
```

| 전환 | 트리거 | 복귀 |
|------|--------|------|
| 메인 → 좌석 상세 | 좌석 탭 | [Close] 또는 외부 탭 |
| 메인 → 카드 입력 | DEAL (Mock) / 수동 폴백 | [Confirm] / [Cancel] |
| 메인 → 금액 입력 | BET / RAISE 버튼 | [BET] / [Cancel] |
| 메인 → Undo | UNDO 버튼 또는 Ctrl+Z | [Close] |
| 메인 → History | 메뉴 > Hand History | [Close] |
| 메인 → 통계 | 메뉴 > Statistics | [Close] |
| 메인 → Settings | [Settings ⚙] | [X] 닫기 |

## 키보드 단축키 요약

| 키 | 동작 | 조건 |
|:--:|------|------|
| N | NEW HAND | IDLE |
| D | DEAL | SETUP_HAND |
| F | FOLD | 베팅 라운드 중 |
| C | CHECK / CALL | biggest_bet 기준 전환 |
| B | BET | biggest_bet == 0 |
| R | RAISE | biggest_bet > 0 |
| A | ALL-IN | 베팅 라운드 중 |
| Ctrl+Z | UNDO | 항상 |
| 0~9 | 좌석 선택 | 항상 |
| Esc | 패널 닫기 | 서브 화면 열려있을 때 |
