## minimal stub 한계

- **실제 Rive 파싱 통과는 보장되지 않는다.** Backend가 magic + size만 검증하면 OK. 본격 Rive 디스플레이 검증이 필요하면 Rive 공식 에디터로 만든 `.riv` 로 교체.
